# Definitions

WELL - The Moonwell Artemis governance token

Liquidity Incentives - WELL tokens that have been allocated for use as rewards to incentivize liquidity growth.

Core - Moonwell Artemis lending protocol (Moonwell: DeFi's Liquidity Well (opens new window)).

DEX LP - StellaSwap’s WELL/GLMR farm (StellaSwap - Leading Moonbeam DEX & DeFi Gateway (opens new window)).
